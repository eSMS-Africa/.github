# eSMS Africa

**Reliable SMS & Email infrastructure for Africa.**

Bulk SMS, OTP delivery, SMPP connectivity, HTTP APIs, and transactional email — powering businesses across 28+ African countries.

---

### Products

**SMS Platform** — Send campaigns, manage contacts, automate OTP delivery, and integrate via REST API or SMPP.

**ESMS Mail** — Transactional email with custom domains, DKIM signing, bounce monitoring, and SMTP relay.
